# Android-Paste  
  
## 安卓依赖dex实现的C++粘贴  
  
需要外部调用(shell和dex)  
仅支持安卓, 可在noroot环境运行  
仅在arm64-v8a android14进行测试  
  
### 小问题  
  
效率很低，粘贴操作要等待一秒左右  
但对此，开发了非阻塞方法  
需要shell环境中存在`app_process64`指令
  
## 对应小白  
  
源文件等在 src/ 文件夹下  
测试器源文件在 test/ 文件夹下  
build.sh是构建测试版本的构建指令  
注意需要安装有g++指令用来编译  

### lib/文件夹  

一个已编译完成的静态库
